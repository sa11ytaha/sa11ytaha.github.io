Cat Facts App

Description:

A simple app to display and search cat facts using the Cat Facts API.

Features:

  • Displays a list of cat facts dynamically.
  • Displays a random cat fact when prompted.
  • Includes a search box for filtering.
  • Reload button to fetch new facts.

Technologies:

  • HTML: Structure.
  • CSS/Bootstrap: Styling and responsiveness.
  • JavaScript: Fetch API for data integration.

API Used:

Base URL: https://cat-fact.herokuapp.com/facts

  • Endpoint 1: /facts - Retrieves a list of cat facts.
  • Endpoint 2: /facts/random - Retrieves a random cat fact.
