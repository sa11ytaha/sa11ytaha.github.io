Lab 3: Responsive Web Design

Making a webpage that works on all devices using Bootstrap.

In this lab, we are learning how to make websites responsive. This means the website will adjust its layout to fit different screen sizes, whether its a laptop or smartphone. We are using Bootstrap to implement this.

Objectives:

  • Learn Bootstrap
  • Use the Grid System
  • Make the page responsive
